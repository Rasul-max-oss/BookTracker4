# 📚 Book Tracker - Трекер книг

## 👤 Автор
**Иван Иванов**

## 📖 Описание программы
Консольное приложение для управления коллекцией книг. Позволяет добавлять, редактировать, удалять и фильтровать книги, а также сохранять данные в JSON.

### Возможности:
- ✅ Добавление книг и аудиокниг
- ✅ Редактирование и удаление
- ✅ Фильтрация по жанру и страницам
- ✅ Отмена действий (СТЕК)
- ✅ История

## Примеры использования
- Выберите действие: 1
- Название: Мастер и Маргарита
- Автор: Булгаков
- Жанр: Роман
- Страниц: 480
- Статус: 1 - To Do
